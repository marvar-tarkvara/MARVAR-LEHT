# MarVariks Homepage

A modern Next.js website for MarVariks web development agency.

## Tech Stack

- **Framework**: Next.js 15 with App Router
- **Runtime**: Bun
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **UI Components**: shadcn/ui
- **Language**: TypeScript

## Getting Started

### Prerequisites

- [Bun](https://bun.sh/) installed on your machine

### Installation

```bash
# Install dependencies
bun install

# Run development server
bun dev

# Build for production
bun run build

# Start production server
bun start
```

### Development

```bash
# Run linting
bun lint

# Run tests
bun test

# Run tests in watch mode
bun test:watch
```
